Tictactoe

Tictactoe is a classic game where people try to get things next to each other. Typically, this is between two people on a three by three grid, with one person using O as their symbol and the other using X. An example of a tictactoe game is a follows:

X O X
O X -
O - X

You can see that a player managed to get three X's in a row, and thus has won.

Notation

Here's how the notation works: there are three characters in total. The first character determines the player (such as X or O), the second character determines the row (starting from the top), and the third character determines the column (starting from the left).
